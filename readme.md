# Project 1
#### To create database
```
CREATE DATABASE users;
```
#### To create table
```
CREATE TABLE users(
  	username VARCHAR(70),
   	password VARCHAR(80),
   	PRIMARY KEY (username)
);
```
#### Starting the webapp
run `app.py`
then go to http://127.0.0.1:5000 to use the app
